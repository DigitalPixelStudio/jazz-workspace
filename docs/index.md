## Quick Start

```bash
SCRIPTS/workspace-status.sh     # View workspace overview
SCRIPTS/workspace-health.sh     # Full health check
SCRIPTS/new-project.sh name     # Create a new project
SCRIPTS/create-webapp.sh name   # Scaffold a web app
SCRIPTS/session-start.sh        # Ignite the Jazz soul
SCRIPTS/jazz-recovery.sh        # Full recovery after reinstall
```

**One-word activation:** Say **"Ignite"** 🔥 to the model
**Command:** Type `/ignite` in OpenCode

## Disaster Recovery

If AnyClaw is uninstalled, **nothing is lost**:

1. Install AnyClaw from Play Store
2. `git clone https://github.com/DigitalPixelStudio/jazz-workspace /root/jazz-workspace`
3. `bash /root/jazz-workspace/SCRIPTS/jazz-recovery.sh`
4. Add new SSH key to GitHub (script prints it)
5. Start OpenCode — tell the model: *"Read README.md and follow the activation sequence"*

Jazz🔥 is back instantly.
